# Relatórios de exemplo

Amostras geradas pelo **motor de relatórios da Suricatoos**, a partir de um **engajamento
fictício** (ACME Corp — aplicação web + Active Directory) criado apenas para demonstração.
Nenhum dado é real.

Servem para mostrar o layout, a identidade visual e os formatos de exportação disponíveis.

## Arquivos

| Relatório | Formatos |
|---|---|
| **Técnico** (achados, evidências, PoC e remediação por tarefa) | [`.docx`](suricatoos-relatorio-tecnico.docx) · [`.pptx`](suricatoos-relatorio-tecnico.pptx) · [`.pdf`](suricatoos-relatorio-tecnico.pdf) |
| **Executivo** (resumo de negócio, gráfico de status, próximos passos) | [`.pptx`](suricatoos-relatorio-executivo.pptx) · [`.pdf`](suricatoos-relatorio-executivo.pdf) |

Os fontes em markdown (`*.md`) também estão na pasta, exatamente como o motor os produz antes da
conversão para Office/PDF.

> ⚠️ Conteúdo meramente ilustrativo. O engajamento, os alvos (`acme.example`) e os achados são
> fictícios e existem só para exemplificar a saída da ferramenta.
